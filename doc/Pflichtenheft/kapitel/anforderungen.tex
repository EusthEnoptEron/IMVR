\section{Sonstige Spezifikationen}

\begin{itemize}
	\item Die Applikation wird aufgeteilt in eine Unity-Applikation und einen Indexer.
	\item Die Unity-Applikation wird erstellt mit Unity 5 Personal.
	\item Der Indexer wird erstellt mit C\# für das .NET Framework 4.
	\item Die Speicherung der Daten erfolgt entweder über eine Serialisierung oder eine SQLite-Datenbank.
	\item Als Entwicklungsumgebung wird Visual Studio 2013 benutzt.
	\item Die Applikation ist auf Englisch.
	\item Es wird eine kurze Anleitung erstellt für die Bedienung der Applikation.
\end{itemize}

\section{Use-Cases}

In dieser Sektion wird eine Auswahl von Use-Cases vorgestellt, die zur Konkretisierung diverses Abläufe dienen soll.

\subsection{UC1: Starten der Applikation}
\label{sec:uc1}

\begin{enumerate}
	\item Der Benutzer startet die Applikation.
	\item Er setzt sein Oculus Rift DK2 mit Leap Motion auf.
	\item \gls{imvr} fordert den Benutzer auf, sich in eine angenehme Sitzposition zu bewegen und die Leertaste zu drücken.
	\item Der Benutzer drückt die Leertaste.
	\item \gls{imvr} gibt dem Benutzer eine Ansicht auf seine Bilder auf, sortiert nach Farbtönen.
\end{enumerate}

\subsection{UC2: Abbruch der Verbindung zur Leap Motion}
\label{sec:uc2}

\begin{enumerate}
	\item Der Benutzer durchläuft UC1.
	\item Die Leap Motion wird nicht (mehr) erkannt.
	\item \gls{imvr} zeigt dem Benutzer eine Warnung als Overlay an.
\end{enumerate}

\subsection{UC3: Beendung der Applikation}
\label{sec:uc3}

\begin{enumerate}
	\item Der Benutzer durchläuft UC1.
	\item Er öffnet das Hauptmenü.
	\item Er wählt den Eintrag für das Schliessen der Applikation.
	\item \gls{imvr} zeigt einen Confirm-Dialog an.
	\item Die Applikation schliesst, falls der Dialog bestätigt wurde. 
\end{enumerate}

\subsection{UC4: Wechseln zur Musikansicht}
\label{sec:uc4}

\begin{enumerate}
	\item Der Benutzer durchläuft UC1.
	\item Er öffnet das Hauptmenü.
	\item Er wählt das Element für Musik.
	\item \gls{imvr} wechselt zur Musikansicht.
\end{enumerate}

Alternative Idee: Klatschen

\subsection{UC5: Hinzufügen eines Medienordners}
\label{sec:uc5}

\begin{enumerate}
	\item Der Benutzer öffnet die Datei ``library.conf''.
	\item Er fügt eine Zeile mit dem Pfad zu seinem Ordner ein.
	\item Er speichert die Datei.
	\item Der Indexer scannt beim nächsten Durchlauf den neuen Ordner.
\end{enumerate}

\subsection{UC6: Abspielen von Hintergrundmusik}

\begin{enumerate}
	\item Der Benutzer durchläuft UC4.
	\item Er wählt einen Artisten aus.
	\item IMVR wechselt zu einer Detailansicht und zeigt alle verfügbaren Alben des Artisten an.
	\item Der Benutzer wählt ein Lied mit den Fingern aus.
	\item IMVR spielt das Lied im Hintergrund ab und zeigt einen Fortschrittsbalken.
\end{enumerate}
