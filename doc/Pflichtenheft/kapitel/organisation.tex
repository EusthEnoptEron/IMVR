\section{Projektmanagement}

\begin{itemize}
	\item Das Projekt wird innerhalb eines \gls{git}-Projektes geführt. Ein privater, zentraler Klon wird auf GitHub abgelegt, welches unter der Adresse \url{https://github.com/EusthEnoptEron/IMVR} erreichbar ist.
	\item Meetings zwischen dem Studenten und dem Betreuer finden jeden zweiten Mittwoch in Biel statt.
	\item Gearbeitet wird durch den Studenten Montags bis Donnerstags an einem Arbeitsplatz im CPVR Labor in Biel.
	\item Das Projekt wird in Scrum-ähnlich mit Sprints vorangetrieben.
\end{itemize}
